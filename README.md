# Early Hypotension Burden and ICU Mortality After Severe Traumatic Brain Injury

This repository contains code and aggregate results for a two-database study of early hypotension burden after severe traumatic brain injury (TBI).

## Study Overview

The study evaluates whether early hypotension burden during the first 24 hours after ICU admission is associated with subsequent ICU mortality among adults with severe TBI.

- Derivation cohort: eICU Collaborative Research Database
- External validation cohort: MIMIC-IV
- Cohort: adult first ICU stays with strict intracranial TBI and GCS <= 8
- Landmark design: patients must survive and remain observed through the first 24 hours after ICU admission
- Primary exposure: time-weighted average MAP deficit below 65 mmHg during ICU hours 0-24
- Prespecified co-exposure: time-weighted average SpO2 deficit below 90 percent during ICU hours 0-24
- Primary outcome: ICU mortality after the 24-hour landmark
- Primary model: eICU logistic regression with locked-coefficient external validation in MIMIC-IV

## Main Results

- Final eICU landmark cohort: N = 777, ICU deaths = 158
- Final MIMIC-IV validation cohort: N = 187, ICU deaths = 26
- Adjusted hypotension burden association: OR = 1.350 per eICU SD, 95 percent CI 1.120-1.629, P = 0.0017
- MIMIC-IV external validation: C-index = 0.729, Brier score = 0.110, calibration slope = 0.687
- Sensitivity analyses using hospital death, MAP <60 mmHg, and 99th-percentile winsorization supported the direction of the main association

Spline analyses are included as exploratory shape analyses. They should not be interpreted as defining actionable clinical cutoffs.

## Data Availability

Patient-level eICU and MIMIC-IV data are not included. These databases require credentialed access and data-use agreements. See `DATA_AVAILABILITY.md` for details.

The included `results/` files are aggregate manuscript tables and figures only.

## Quick Start

To inspect the published aggregate results, open the files under `results/`.

To rerun the code, install Python dependencies:

```powershell
python -m pip install -r requirements.txt
```

Full reproduction requires local PostgreSQL databases for eICU and MIMIC-IV and the derived intermediate files generated by the SQL runners. See `REPRODUCE.md`.
