# Package Manifest

## Code

- `02_code/sql/`: SQL scripts for cohort flow, sampling QC, burden construction, and covariate extraction.
- `02_code/python/`: Python scripts for model fitting, validation, sensitivity analyses, QC, and final figure/table generation.
- `02_code/run_*.ps1`: PowerShell runners with generic `python` and `psql` defaults.

## Results

- `results/tables/csv/`: aggregate final manuscript and supplementary tables.
- `results/tables/md/`: Markdown versions of aggregate final tables.
- `results/tables/docx/`: final DOCX manuscript table files.
- `results/figures/svg/`: editable vector figure exports.
- `results/figures/png/`: 600 dpi PNG figure exports.
- `results/figures/tiff/`: 600 dpi TIFF figure exports.

## Documentation

- `README.md`: public project overview.
- `DATA_AVAILABILITY.md`: source-data and redistribution rules.
- `REPRODUCE.md`: local reproduction steps.
- `CODEBOOK.md`: key variables and model definitions.
- `docs/FINAL_DECISION_NOTE.md`: final artifact decision summary.
- `docs/FINAL_ARTIFACT_QA_SUMMARY.md`: final artifact QA summary.
